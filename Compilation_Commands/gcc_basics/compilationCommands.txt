Compile:    gcc programName.c
Run:        ./a.out

Compile:    gcc -o executableName programName.c
Run:        ./executableName
Note: "executableName" and "filename" can be used interchangeably.

Compile:    gcc -std=c11 -Wall -fmax-errors=10 -Wextra programName.c -o executableName
Run:        ./executableName

Alternately,

Compile:    gcc -std=c11 -Wall -fmax-errors=10 -Wextra -o executableName programName.c 
Run:        ./executableName

C-standard: 				-std=c11
Show all warnings:		        -Wall
Stop compiling after 10 errors:		-fmax-errors=10
Show extra compiler warnings:		-Wextra









Command to terminate a program in terminal:

PRESS control AND c SIMULTANEOUSLY ON KEYBOARD
